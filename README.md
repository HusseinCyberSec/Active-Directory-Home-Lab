# Active Directory Home Lab

## Overview
A hands-on home lab simulating an enterprise IT environment using Oracle VirtualBox. 
This project documents the setup, configuration, and administration of Windows Server 
with Active Directory Domain Services (AD DS).

## Tools & Technologies
- Oracle VirtualBox
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- PowerShell *(coming soon)*

## Lab Progress
- [x] Windows Server installed on VirtualBox
- [x] Administrator account configured
- [ ] Active Directory Domain Services installed
